# Stars Tape Website

Ovo je demo veb sajt napravljen za lokalni biznis "Stars Tape".  
Sajt je responsive i prilagođen mobilnim uređajima.  

## Funkcionalnosti
- Kontakt forma (za upite klijenata)  
- Responsive dizajn za telefone i tablete  
- Jednostavna struktura, lako za nadogradnju  

## Tehnologije
- HTML  
- CSS  
- JavaScript  

## Napomena
Trenutno sajt služi za prikaz i kontakt; kasnije se može dodati opcija online naručivanja.
